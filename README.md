## :pushpin: 구현 기능

> # React Native (expo) 로 만든 날씨앱 
> * 앱이 실행되자마자 유저는 AsyncStorage를 이용한 로컬에 저장된 자료를 불러오고, 앱이 로딩되면서 날씨가 불러와진다.
> * 오늘의 날씨를 image 와 text로 보여지고 풍량, 습도, 일출시간을 보여준다.
> * daily forecast : 7일간의 일기예보를 사용자에게 알려준다.
